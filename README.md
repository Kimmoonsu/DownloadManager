# File Download Project

> 서버로부터 50개의 파일 URL을 받아서 사용자에게 보여주고, 그 중 여러개를 선택해서 다운로드 받을수 있는 기능을 안드로이드앱으로 구현하기

- 지원 기능
    - 파일 다운로드
    - 파일 모두 선택 (Select ALL)
    - 현재상태 저장
    - 파일 다운 성공/실패 표시

- 사용 기술 및 오픈소스
    - MVVM 패턴
    - DownloadManager
    - Gson
    - SharedPreference
    - NetworkCallback
    - RecyclerView
